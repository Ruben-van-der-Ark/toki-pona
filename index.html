Text Formatting Information:

You can use a few types of formatting:
First, you can use a dash to make the two glyphs in sitelen pona connect.
The latin alphabet version will automatically replace a dash with a space.
Example: toki-pona becomes:
toki-pona
toki pona
(Keep in mind that the version of the font I'm using (linja pona version 4.9) automatically generates these combined glyphs, so they might not always look good.)

Second, you can use the standard linja pona syntax to spell out foreign words.
Example: toki [_ijo_ni_li_ike] becomes:
toki [_ijo_ni_li_ike]
toki Inli

Lastly, as is standard in toki pona, every letter in the latin alphabet version is automatically converted to lowercase (unless you use the format specified above).
Example: tOKI ni lI JAKI MutE becomes:
toki ni li jaki mute
toki ni li jaki mute

This website uses the font linja pona (version 4.9)
